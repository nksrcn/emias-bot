# ЕМИАС Бот — автозапись с Telegram авторизацией

## Как это работает

```
Первый запуск / истечение сессии:
  Telegram → кнопка "Войти в ЕМИАС"
  → Открываете ссылку в браузере
  → Логинитесь один раз
  → Бот продолжает сам

Обычная работа:
  🟢 Каждые 5 минут проверяет талоны
  🔴 07:25–07:45 МСК — каждые 5 секунд
  ✅ Нашёл талон → записался → уведомил в Telegram
```

---

## Шаг 1 — Telegram бот

1. Напишите **@BotFather** → `/newbot`
2. Скопируйте токен: `7123456789:AAF...`
3. Напишите **@userinfobot** → скопируйте ваш `id`

## Шаг 2 — GitHub

Создайте репозиторий, загрузите все файлы:
- `main.py`
- `bot.py`
- `auth_server.py`
- `requirements.txt`
- `Dockerfile`
- `railway.toml`

## Шаг 3 — Railway

1. [railway.app](https://railway.app) → New Project → GitHub repo
2. Settings → Networking → Generate Domain (запомните домен!)

## Шаг 4 — Переменные окружения (Variables)

| Переменная              | Значение                              |
|-------------------------|---------------------------------------|
| `EMIAS_LOGIN`           | Ваш логин ЕМИАС                       |
| `EMIAS_PASSWORD`        | Ваш пароль ЕМИАС                      |
| `TG_TOKEN`              | Токен от @BotFather                   |
| `TG_CHAT_ID`            | Ваш chat_id от @userinfobot           |
| `RAILWAY_PUBLIC_DOMAIN` | Домен Railway (без https://)          |
| `CHECK_INTERVAL_NORMAL` | `300` (5 минут, можно менять)         |
| `CHECK_INTERVAL_ACTIVE` | `5` (5 секунд в активном окне)        |

> `RAILWAY_PUBLIC_DOMAIN` — Railway подставляет автоматически если включить
> Generate Domain в настройках сервиса.

## Что получите в Telegram

- 🤖 Бот запущен
- 🔑 Кнопка логина (если нужна авторизация)
- 🔴 Переключение в активный режим в 07:25
- 🟢 Возврат в обычный режим в 07:45
- ✅ Запись подтверждена (с датой и временем)
- ⚠️ Ошибки если что-то пошло не так
